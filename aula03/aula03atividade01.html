Atividade 01 - Aula 03

A atividade consiste em criar um programa que solicita dois números ao usuário e exibe a soma deles.

Utilizando de variáveis e prompt para entrada de dados.

E o resultado da soma é exibido em um alerta.

Exemplo de código JavaScript para a atividade:

let num1 = parseFloat(prompt('Informe o primeiro número: '));

let num2 = parseFloat(prompt('Informe o segundo número: '));

let soma = num1 + num2;
